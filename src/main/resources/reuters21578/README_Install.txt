To test TopicGrouperJ on the Reuters 21578 data set:
* Download the document collection from here: http://www.daviddlewis.com/resources/testcollections/reuters21578/
* Unzip it and place all its files in this folder.
* Ensure that ./lib/lingpipe-3.9.3.jar is included in your Java classpath.
* Run e.g. org.hhn.topicgrouper.figures.MindMapDemoReuters in the project directory.
* Find related output in ./target.
